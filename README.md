# 📦 Amazon Sales EDA Dashboard

[![Streamlit App](https://img.shields.io/badge/Streamlit-FF4B4B?style=for-the-badge&logo=Streamlit&logoColor=white)](https://streamlit.io/)
[![Python 3.10+](https://img.shields.io/badge/Python-3.10+-3776AB?style=for-the-badge&logo=python&logoColor=white)](https://www.python.org/downloads/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow?style=for-the-badge)](https://opensource.org/licenses/MIT)

A premium, interactive **Exploratory Data Analysis (EDA)** dashboard designed to provide deep insights into Amazon e-commerce sales. Built with a high-performance Python stack.

---

## 🚀 Key Features

*   **📊 Executive KPIs**: Instant view of Total Revenue (formatted in Millions), Average Rating, and Order Volume.
*   **🎨 Dynamic Theming**: Toggle between **Modern**, **Vibrant (Dark Mode)**, and **Corporate** styles instantly via the sidebar.
*   **🏆 Performance Analytics**: Visual breakdown of revenue by product category and geographic region.
*   **🌍 Market Insights**: Deep dives into regional customer satisfaction and payment method preferences.
*   **💳 Transactional Trends**: Comparative analysis of payment methods across different global territories.
*   **🔍 Interactive Filtering**: Real-time updates across all charts based on Category and Region selections.
*   **📄 Data Explorer**: On-demand access to the underlying filtered dataset.

---

## 🛠️ Tech Stack

- **Frontend/App**: [Streamlit](https://streamlit.io/)
- **Data Manipulation**: [Pandas](https://pandas.pydata.org/)
- **Visualizations**: [Plotly Express](https://plotly.com/python/plotly-express/)
- **Theming**: Custom CSS Injection & Plotly Templates

## ⚙️ Installation & Setup

1. **Clone the Repository**
   ```bash
   git clone <your-repo-url>
   cd "Amazon_sales Project EDA"
   ```

2. **Create a Virtual Environment**
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # On Windows use: .venv\Scripts\activate
   ```

3. **Install Dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Launch the Dashboard**
   ```bash
   streamlit run app.py
   ```

---

## 📂 Project Structure

- `app.py`: Main dashboard application logic and UI.
- `Data/`: Directory containing the `amazon_sales_dataset.csv`.
- `.streamlit/config.toml`: Base theme and server configurations.
- `requirements.txt`: Python package dependencies.
